# 基于ssm的轻语音乐网网站

#### 介绍
基于ssm的轻语音乐网网站，前端采用html+bootstrap，主要分两端Web端与管理后端，代码规范整洁，技术选型也非常经典，合适初学java编程者学习以及计算机专业学生。


#### 软件架构
前端：html | jquery | bootstrap  
后端：spring | springmvc | mybatis  
环境：jdk1.8 | maven | mysql | tomcat
